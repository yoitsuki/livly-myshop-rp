# データソース

## マイショップ開催回データ

`src/lib/shopPeriods.ts` の `SHOP_ROUNDS` に格納している各回の開催期間は、リヴリー攻略まとめサイト「**リヴリーガイド**」の以下のページから引用しました。

- リヴリー マイショップ 開催履歴: <https://livly-guide.com/livly-myshop/>

> 日付・時刻はすべて JST。開催期間は `start`〜`end` の epoch ms で保持し、`resolveShopPeriod()` で確認日時 (`checkedAt`) を「開催中 / 最終日」に分解しています。

## 注意

このページの内容は記録用メモです。アプリ側ではデータのみを利用しており、画面に出典は掲示していません。
新しい回が追加された場合は `SHOP_ROUNDS` に追記し、必要なら本ファイルも更新してください。
